1. Classification Loss (CE)

Mục đích: Predict có object hay không
Target: Binary (0/1) cho mỗi query
Loss: Binary cross-entropy
Matching: Hungarian assignment

2. Bounding Box Loss (Bbox)

Mục đích: Regression tọa độ bbox chính xác
Target: Ground truth coordinates (x1,y1,x2,y2)
Loss: Smooth L1 loss
Note: Chỉ apply cho matched queries

3. Similarity Loss (Sim)

Mục đích: Open-vocab classification
Target: 1 cho matched regions, 0 cho background
Loss: BCE between region-text similarity
Innovation: Enables zero-shot detection

4. Hungarian Matching

Mục đích: Optimal assignment predictions ↔ targets
Cost: Classification + Bbox + GIoU costs
Algorithm: scipy.optimize.linear_sum_assignment
Result: Bipartite matching indices

💡 Key Insight:
3 losses work together:

CE: Detect objectness
Bbox: Localize accurately
Sim: Classify with language
